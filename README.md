### 分科校系分數查詢系統
https://university-ast-inquire-system.streamlit.app/

---
### 功能介紹
提供查詢今年 (115) 和前三年 (112~114) 的校系資訊，  
包括 學測檢定標準、採計科目、歷年分數、歷年分數平均；  
除此之外，你可以輸入學測分數，會幫你計算你分科還需要考幾分才會上。  

---
### 操作說明
1. 搜尋校系  
   只輸入校名，例如"台大"、"成大"；會輸出所有該大學的科系。  
   只輸入系名，例如"電機"、"醫學系"；會輸出所有有該科系的大學。  
   你可以 換行 一次搜尋多個校系。
   
2. 學校清單  
   勾選學校清單，輸出結果只會有清單上的學校。  
   你可以用來快速搜尋你想要的科系。

3. 計算所需分數  
   輸入你的學測分數，
   會幫你計算前幾年每個科系你分科需要考的分數。
